# Ernest Okpako — Portfolio

A high-end, minimalist portfolio website built with clean HTML5, CSS3, and Vanilla JavaScript. This project showcases my work as a Designer and Developer, featuring real-world projects, deep-dive blog posts, and a responsive design.

## 🚀 Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Micro-Animations:** Smooth scroll-reveal effects and interactive hover states using CSS and Intersection Observer API.
- **Dynamic Content:** Custom JavaScript logic to load project case studies and blog posts using URL parameters.
- **Premium Aesthetics:** Modern typography (Geist), dark/light adaptive elements, and professional MacBook mockups for work presentation.
- **SEO Optimized:** Semantic HTML5 structure with proper meta tags for search visibility.

## 📂 Project Structure

- `index.html`: Homepage with featured projects, skills, and CTA.
- `about.html`: Detailed professional background and experience.
- `work.html`: Full gallery of real-world projects.
- `blog.html`: List of deep-dive articles on design and development.
- `project.html`: Dynamic template for project case studies.
- `blog-post.html`: Dynamic template for reading long-form blog content.
- `css/style.css`: Centralized design system with CSS variables.
- `js/main.js`: Core interactivity (nav, scroll reveal, mobile menu).
- `js/project.js`: Data store and logic for project case studies.
- `js/blog.js`: Data store and logic for blog articles.
- `images/`: High-resolution MacBook mockups and project assets.

## 🛠️ Built With

- **Structure:** HTML5 (Semantic)
- **Styling:** CSS3 (Vanilla, Custom Properties)
- **Logic:** JavaScript (ES6+, No Frameworks)
- **Fonts:** Geist (Google Fonts)
